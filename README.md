# 🌌 Colliding Black Holes Simulator - Official Release

Welcome to the very first official public release of the **Colliding Black Holes Simulator (CBHS)**! This application invites you to explore the fascinating physics behind black hole mergers and the discovery of gravitational waves in a highly interactive, educational environment.

## ✨ Key Features
* **Dynamic Physics Engine:** Experience real-time simulations of binary black hole inspirals and mergers, featuring accurate gravitational waveform rendering and smooth ringdowns.
* **Interactive Data Analysis:** A dual-window OS-style system featuring a 'Cosmic View' simulation alongside a dedicated 'LIGO Detector' module.
* **Meet "Observy":** Your personal, animated observatory guide! Observy reacts to the simulation in real-time, offering fun insights and teaching core physics concepts dynamically.
* **Wave Sonification:** Experience the cosmos by listening to the iconic "chirp" of gravitational waves converted into audible frequencies.
* **Bilingual Support:** Full, seamless UI support for both English and Slovak languages.

## 📥 Download & Usage Instructions

No installation is required! The simulator is designed to be completely portable (Plug & Play). Please download the exact file for your operating system from the **Assets** list at the bottom of this page.

### 🪟 Windows (.exe)
1. Download the `CBHS_v5_0_Win.exe` file.
2. Double-click the file to launch the simulator instantly!

### 🐧 Linux (Standalone Binary)
1. Download the `CBHS_v5_0_Linux` file.
2. Because the file was downloaded from the internet, Linux requires you to grant it execution permissions before running.
   * **UI Method:** Right-click the downloaded file -> `Properties` (or `Show details`) -> `Permissions` tab -> Check the box for **"Allow executing file as program"** (or select `Read and execute`). Close the window and double-click the file to launch!
   * **Terminal Method:** Open a terminal in the folder where the file is downloaded and run:
     `chmod +x CBHS_v5_0_Linux`
     Then launch it by typing:
     `./CBHS_v5_0_Linux`
