# 🌌 Colliding Black Holes Simulator (CBHS)

An interactive, educational simulator that lets you explore the physics of black hole mergers and gravitational waves.

---

## ✨ What is CBHS?

CBHS is a desktop application built around the science behind **LIGO's groundbreaking gravitational wave detections**. You control the parameters, watch the collision unfold, and hear the universe chirp. Each version brings new physics, better visuals, and more real-world data.

Whether you're a curious student, a physics enthusiast, or just someone who thinks black holes are cool – this is for you.

---

## 🚀 Key Features

- **Physics-based simulation** of binary black hole inspirals, mergers, and ringdowns
- **GPU-accelerated 2D wave field** – pre-generated for smooth, lag-free playback
- **GWOSC Archive integration** – load and visualize real gravitational wave events (GW150914 and more)
- **Realistic inclination / viewing angle** – see how the system looks from any angle in 3D
- **Wave sonification** – hear the iconic gravitational wave "chirp" converted to audio
- **Interactive physics display** – live chirp mass, event horizon, peak frequency, and merger duration
- **Bilingual UI** – full English and Slovak language support

## 📥 Download & Run

No installation required. Download the build for your OS from the **Releases** page and run it directly.

### 🪟 Windows
1. Download the `.exe` file from the latest release
2. Double-click to launch

> ⚠️ **SmartScreen warning**: Windows may show "Windows protected your PC" because the app is unsigned. This is normal for indie apps.
> Click **"More info"** → **"Run anyway"**. Verified safe by beta testers.

### 🐧 Linux
1. Download the Linux binary from the latest release
2. Grant execute permission:
   - **UI**: Right-click → Properties → Permissions → *Allow executing as program*
   - **Terminal**:
     ```bash
     chmod +x CBHS_Linux
     ./CBHS_Linux
     ```
